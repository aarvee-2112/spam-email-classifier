# 📧 Spam Email Classifier

An AI-powered Spam Email Classifier built using Python, Scikit-learn, TF-IDF, Naive Bayes, and Streamlit.

---

## 🚀 Features

- Detects Spam and Not Spam messages
- Text preprocessing using NLTK
- TF-IDF Vectorization
- Multinomial Naive Bayes Machine Learning Model
- Interactive Streamlit Web Application

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Streamlit
- Git & GitHub

---

## 📊 Model Performance

- Accuracy: **97.29%**
- Precision: **99.16%**
